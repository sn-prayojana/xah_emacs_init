All Windows 10 window logo key shortcuts.

Change Windows Size

  • 【❖Window+↑】 Maximize window.
  • 【❖Window+↓】 Restore window size.
  • 【❖Window+Shift+↑】 Maximize window vertically.
  • 【❖Window+Shift+↓】 Restore vertical size or hide window.
  • 【❖Window+←】 or 【❖Window+→】 Align the window to the side of the screen, maximizing it vertically.
  • 【❖Window+Shift+←】 or 【❖Window+Shift+→】 Move the window to the next or previous monitor.

    • Show/Hide

  • 【❖Window+D】 toggle show desktop
  • 【❖Window+M】 Minimize all windows. 【❖Window+Shift+M】 restore
  • 【❖Window+Home】 hide all other windows. press again to show

    • Swich/Launch App

  • 【❖Window+Tab】 show app history.
  • 【❖Window+T】 go thru apps on taskbar.
  • 【❖Window+1】, 【❖Window+2】, run the app on taskbar. with Shift, start a new instance.
  • 【❖Window+E】 Open Explorer (view folders).

    • misc

  • 【❖Window+L】 Lock desktop.
  • 【❖Window+B】 Select the first icon in the Notification Area.
  • 【❖Window+R】 Open the Run dialog.
  • 【❖Window+G】 show Windows gadgets.

    • System

  • 【❖Window+Pause】 Open System Properties.
  • 【❖Window+U】 open display settings.
  • 【❖Window+Spacebar】 switch language.
  • 【❖Window+Spacebar】 View desktop. (activate Aero Peek).
  • 【❖Window+P】 Choose a presentation display mode (set screen to clone on 2nd monitor, etc.).
  • 【❖Window++】 Zoom-in screen. 【❖Window+-】 Zoom-out screen. 【❖Window+Escape】 Cancel magnification.
